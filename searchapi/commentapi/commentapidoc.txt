PURPOSE

Build a service like:

YouTube Comments

Reddit Comments

Users can:

✓ Create Comment
✓ Edit Comment
✓ Get Comments
✓ Delete Comment
✓ Flag Comment

--------------------------------------------------------------------
FUNCTIONAL REQUIREMENTS
--------------------------------------------------------------------

Create Comment
→ User can post comment on a post

Update Comment
→ User can edit own comment

Get Comment(s)
→ Retrieve comments for a post

Delete Comment
→ Remove own comment

Flag Comment
→ Report inappropriate content

Optional Features

Reply To Comment
Like Comment
Pin Comment
Mention User
Media Comments

--------------------------------------------------------------------
NON-FUNCTIONAL REQUIREMENTS
--------------------------------------------------------------------

Scalability
→ Millions of comments/day

Availability
→ Service always accessible

Security
→ Only authorized actions

Reliability
→ No data loss

Low Latency
→ Comments visible quickly

  "comment":"Amazing Post!"
}

  "commentId":"789",
  "status":"CREATED"
}

      |

      |
